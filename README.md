# SyncPlay-LAN

SyncPlay-LAN is a Windows-first Java desktop application that allows a user to play a local video file and stream its audio (and optionally video) over the local Wi-Fi network so multiple nearby devices can connect and listen in sync using their own headphones.

## Features
- **Local Network Sync**: Play video on PC, listen on phone/tablet via browser.
- **Offline First**: No internet connection required.
- **Privacy Focused**: No clouds, no accounts, no tracking.
- **Cross-Device**: Client runs in any modern web browser.

## Getting Started

### Prerequisites
- Java 17 or higher
- VLC Media Player installed (for VLCJ support)

### Running
1. Clone the repository.
2. Run `./gradlew run`.
