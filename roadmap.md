# Roadmap

## Phase 1: MVP (Current)
- [ ] Basic project setup (Gradle, JavaFX, VLCJ)
- [ ] Video playback on Desktop
- [ ] Serving static HTML client
- [ ] Audio streaming via HTTP
- [ ] Basic synchronization via WebSocket

## Phase 2: Refinement
- [ ] Latency compensation logic (NTP-like sync)
- [ ] UI Polish (Dark mode, custom controls)
- [ ] Connect via QR Code

## Phase 3: Advanced Features
- [ ] Video mirroring (MJPEG or HLS)
- [ ] Playlist support
- [ ] Subtitle support
