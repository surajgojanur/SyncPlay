# SyncPlay-LAN Brand Guidelines

## Brand Personality
- Quiet
- Technical
- Trustworthy
- Utility-first

## Colors
- Background: #0F1115
- Surface: #1A1D24
- Primary Accent: #4DA3FF
- Success: #3DDC97
- Error: #FF5C5C
- Text Primary: #E6E6E6
- Text Secondary: #A0A0A0

## Typography
- UI Font: Inter / System UI fallback
- Headings: Semi-bold
- Body: Regular

## UI Rules
- No gradients
- No animations unless functional
- Icons must be outline style
- Status colors must be semantic

## Logo Usage
- Logo always top-left
- Never stretch
- Minimum padding: 16px
